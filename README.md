# Python Study Material 🐍📚

Welcome to the **Python Study Material** repository! This repository serves as a comprehensive, structured guide for learning Python, ranging from foundational concepts to advanced programming paradigms, data structures, and hands-on projects.

Whether you are starting from scratch or brushing up on advanced concepts like object-oriented programming, concurrency, or algorithms, this repository is organized to help you master Python step-by-step.

## 📂 Repository Structure

## 🚀 Topics Covered

### 1. Core Python Foundations

* **Basics:** Syntax, dynamic typing, type casting, arithmetic/logical operations.
* **Control Flow:** `if-elif-else` conditional execution, `while` loops, and `for` loops with `range()`.

### 2. Built-in Data Structures

* Deep dive into sequences: Mutability vs. Immutability.
* Efficient utilization of Dictionaries and Sets for $O(1)$ lookups.
* Memory layout and built-in methods for Lists and Tuples.

### 3. Functional Programming

* Writing modular, reusable code using functions (`def`).
* Understanding `*args` and `kwargs` for variable-length arguments.
* Anonymous functions via `lambda`, mapping, filtering, and reducing.

### 4. Object-Oriented Programming (OOP)

* Real-world modeling using Classes and Instances.
* Implementing the 4 pillars: **Inheritance**, **Polymorphism**, **Encapsulation**, and **Abstraction**.
* Dunder/Magic methods (`__init__`, `__str__`, `__repr__`, etc.).

### 5. Advanced Mechanics

* Custom **Decorators** for structural modification of function behavior.
* **Generators** and the `yield` keyword for memory-efficient data streaming.
* Exception handling workflows using `try-except-else-finally`.

## 🛠️ Getting Started

To get a local copy of this repository up and running, follow these simple steps.

### Prerequisites

Make sure you have Python 3.x installed on your machine. You can download it from [python.org](https://www.python.org/).

Verify your installation:

```bash
python --version

```

### Installation & Usage

1. Clone the repository:
```bash
git clone [https://github.com/username/Python-Study-Material.git](https://github.com/username/Python-Study-Material.git)

```


2. Navigate into the repository directory:
```bash
cd Python-Study-Material

```


3. Navigate into any specific topic folder and run the scripts:
```bash
cd 01_Basics
python variables.py

```
